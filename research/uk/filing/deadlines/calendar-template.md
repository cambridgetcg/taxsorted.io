## Filing Checklist

### Before Period End

- [ ] Reconcile accounts
- [ ] Review VAT
- [ ] Check PAYE submissions complete
- [ ] Prepare year-end figures

### After Period End

- [ ] File VAT return (within 1m+7d)
- [ ] Prepare accounts
- [ ] File CH accounts (9m)
- [ ] File CT600 (12m)
- [ ] Pay CT (9m+1d)

### Tax Year End (5 April)

- [ ] Final FPS submitted
- [ ] P60s prepared
- [ ] P11D information gathered
- [ ] SA return data ready
