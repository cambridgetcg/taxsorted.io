# MVP Frontend: Complete Path Map

> **Purpose**: Visual map of all user journeys and screens for TaxSorted.io MVP
> **Design Date**: 2026-02-01
> **Framework**: Next.js 15 + TypeScript + Tailwind CSS

---

## MVP Scope Definition

### Core Purpose
**TaxSorted.io** helps UK taxpayers (individuals and businesses) track and submit their tax filings on time.

### MVP Value Proposition
```
"See all your filing deadlines in one place. Never miss a deadline again."
```

### MVP Feature Set (Prioritized)

| Priority | Feature | Description |
|----------|---------|-------------|
| P0 | Entity Creation | Create/manage taxpayer profiles |
| P0 | Obligation Dashboard | View all filings and deadlines |
| P0 | Filing Status | Track submission lifecycle |
| P1 | VAT MTD Submission | Submit VAT returns via API |
| P1 | Calendar View | Visual deadline calendar |
| P2 | Document Export | Export data for portal filing |
| P2 | Reminders | Email/notification alerts |
| P3 | Multi-entity | Manage multiple entities |
| P3 | Agent Portal | Accountant multi-client view |
