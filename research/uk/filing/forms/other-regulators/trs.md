# Trust Registration Service (TRS)

> **Authority**: HMRC
> **Last Updated**: 2026-02-01

---

## Overview

| Aspect | Detail |
|--------|--------|
| **Full Name** | Trust Registration Service |
| **Purpose** | Register trusts and maintain beneficial ownership |
| **Who Files** | Trustees of most UK trusts |
| **Frequency** | Initial registration + annual updates |
| **Method** | Online only (GOV.UK) |

---

## Background

### Why TRS Exists

Implements EU Anti-Money Laundering Directives (4MLD, 5MLD):
- Beneficial ownership transparency
- Anti-money laundering compliance
- Tax compliance

---

## Who Must Register

### Taxable Trusts (Always)

| Trust Type | Register? | When |
|------------|-----------|------|
| **Trust with UK tax liability** | Yes | Within 90 days of becoming taxable |
| **Express trust with CGT** | Yes | 90 days |
| **Trust with IHT liability** | Yes | 90 days |
| **Trust with Income Tax** | Yes | 90 days |
| **Trust with SDLT** | Yes | 90 days |

### Non-Taxable Trusts (5MLD)

Since October 2020, most trusts must register even without UK tax liability:

| Trust Type | Register? |
|------------|-----------|
| **Express trusts (UK)** | Yes |
| **Express trusts (non-UK with UK property/business relationship)** | Yes |
| **Trusts with UK land** | Yes |

---

## Trusts Exempt from Registration

### Excluded Trust Types

| Trust Type | Exempt? |
|------------|---------|
| **Bare trusts** | Yes (unless taxable) |
| **Life insurance trusts** | Yes (until claim) |
| **Registered pension schemes** | Yes |
| **Authorised unit trusts** | Yes |
| **Charitable trusts** | Yes (registered with CC) |
| **Will trusts** | Yes (for 2 years after death) |
| **Co-ownership** | Yes (if no trustees) |
| **Bank accounts (minor)** | Yes |
| **SIPP/SSAS** | Yes |
| **Pilot trusts (pre-Oct 2020, <£100)** | Yes |

---

## What Must Be Registered

### Trust Details

| Information | Required |
|-------------|----------|
| **Trust name** | Yes |
| **Date established** | Yes |
| **Type of trust** | Yes |
| **Country of administration** | Yes |
| **Trust deed** | Reference (not upload) |

### Beneficial Owners

| Person | Details Required |
|--------|------------------|
| **Settlor(s)** | Name, DOB, address, NI number |
| **Trustee(s)** | Name, DOB, address, NI number |
| **Beneficiary/ies** | Name, DOB, address (or class description) |
| **Protector** | If applicable |

### Class of Beneficiaries

If beneficiaries not identifiable:
- Describe the class
- e.g., "grandchildren of settlor"

---

## Registration Deadlines

### Taxable Trusts

| Trigger | Deadline |
|---------|----------|
| **New trust with tax liability** | 90 days from creation |
| **Existing trust becomes taxable** | 90 days |
| **By 31 January** | If filing SA900 |

### Non-Taxable Trusts

| Situation | Deadline |
|-----------|----------|
| **Existing trusts (pre-6 Oct 2020)** | Were required by 1 September 2022 |
| **New trusts (post-6 Oct 2020)** | 90 days from creation |

---

## Annual Updates

### Declaring No Changes

| Aspect | Requirement |
|--------|-------------|
| **Frequency** | Annual |
| **Deadline** | 31 January (if taxable) or within 90 days of change |
| **What** | Confirm details still accurate |

### When Changes Occur

Must update within 90 days of:
- Change of trustee
- Change of beneficiary
- Change of trust address
- Death of settlor/trustee/beneficiary
- Trust assets change significantly

---

## How to Register

### Online Only

| Aspect | Detail |
|--------|--------|
| **Portal** | GOV.UK Trust Registration Service |
| **Account** | Government Gateway |
| **Who registers** | Lead trustee or agent |
| **Fee** | Free |

### Agent Registration

Tax agents can:
- Register trusts for clients
- Update trust details
- Maintain multiple trusts

## Unique Taxpayer Reference (UTR)

### Taxable Trusts

| Situation | UTR? |
|-----------|------|
| **Trust with tax liability** | Issued by HMRC |
| **Need to file SA900** | UTR required |

### Non-Taxable Trusts

| Situation | UTR? |
|-----------|------|
| **No tax liability** | No UTR issued |
| **Reference number** | TRS provides reference |

---

## Penalties

### Failure to Register

| Offence | Penalty |
|---------|---------|
| **Failure to register** | Up to £5,000 |
| **Inaccurate information** | Up to £5,000 |
| **Failure to update** | Up to £5,000 |

### Linked to SA900

If trust has SA900 liability:
- TRS registration is condition for filing
- Cannot file SA900 without TRS registration

---

## Access to Trust Information

### Who Can Access

| Requestor | Access Level |
|-----------|--------------|
| **HMRC** | Full access |
| **Law enforcement** | Full access |
| **Regulated businesses** | Limited (AML checks) |
| **Public** | Limited (legitimate interest - rarely granted) |

### Legitimate Interest Access

Very restricted:
- Must demonstrate legitimate interest
- Anti-money laundering purposes
- Generally refused for general enquiries

## Trust Types and TRS

### Express Trusts

| Type | Register on TRS? |
|------|------------------|
| **Discretionary trust** | Yes |
| **Interest in possession** | Yes |
| **Bare trust** | Usually no (unless taxable) |
| **Life interest trust** | Yes |
| **Accumulation trust** | Yes |

### Non-Express Trusts

| Type | Register on TRS? |
|------|------------------|
| **Resulting trust** | No |
| **Constructive trust** | No |
| **Statutory trust** | No |

---

## Overseas Trusts

### When UK Registration Required

| Situation | Register? |
|-----------|-----------|
| **UK resident trustee** | Yes |
| **UK source income** | Yes |
| **UK property** | Yes |
| **UK business relationship** | Yes |
| **No UK connection** | No |

---

## Death and Estates

### Will Trusts

| Period | Register? |
|--------|-----------|
| **First 2 years after death** | No (estate administration) |
| **After 2 years** | Yes (if still continuing) |
| **IHT purposes only** | Register if paying IHT |

### Deceased Persons

Must update TRS within 90 days of:
- Settlor death
- Trustee death
- Beneficiary death

---

## Common Scenarios

### Family Trust

| Scenario | Action |
|----------|--------|
| **Parents create trust for children** | Register within 90 days |
| **Annual income >£500** | Also get UTR, file SA900 |
| **No income** | Just TRS registration |

### Property Trust

| Scenario | Action |
|----------|--------|
| **Trust owns UK property** | Must register |
| **Rental income** | Also SA900 required |
| **Sale (CGT)** | Report on SA900 |
