# Charity Commission Annual Return

> **Authority**: Charity Commission for England and Wales
> **Last Updated**: 2026-02-01

---

## Overview

| Aspect | Detail |
|--------|--------|
| **Full Name** | Annual Return |
| **Purpose** | Update charity register, report activities |
| **Who Files** | All registered charities (income thresholds apply) |
| **Frequency** | Annual (within 10 months of financial year end) |
| **Method** | Online only |

---

## Who Must File

### Income Thresholds

| Charity Income | Annual Return Required? |
|----------------|------------------------|
| **Under £10,000** | Yes (basic update) |
| **£10,000 - £25,000** | Yes (standard) |
| **£25,000 - £500,000** | Yes (+ accounts) |
| **£500,000 - £1m** | Yes (+ detailed) |
| **Over £1m** | Yes (full return) |

### All Registered Charities Must

- Update basic information annually
- Confirm contact details
- Report serious incidents (if any)

---

## What's Reported

### Basic Information (All Charities)

| Section | Content |
|---------|---------|
| **Trustees** | Names and appointment dates |
| **Contact details** | Address, email, website |
| **Financial year end** | Confirm or change |
| **Income/expenditure** | Total figures |

### Additional Reporting (£25k+)

| Section | Content |
|---------|---------|
| **Activities** | What charity did this year |
| **Achievements** | Impact and outcomes |
| **Financial review** | Reserves, investments |
| **Risk management** | How risks managed |

### Full Return (£1m+)

| Section | Content |
|---------|---------|
| **Governance** | Detailed trustee information |
| **Safeguarding** | Policies and incidents |
| **Fundraising** | Methods, compliance |
| **Overseas activities** | If applicable |

---

## Deadline

### 10 Months After Financial Year End

| Financial Year End | Deadline |
|-------------------|----------|
| 31 March 2026 | 31 January 2027 |
| 31 December 2025 | 31 October 2026 |
| 30 June 2026 | 30 April 2027 |

---

## How to File

### Online Only

| Aspect | Detail |
|--------|--------|
| **Portal** | Charity Commission online services |
| **Account** | Charity Commission account |
| **Who can file** | Named contact or authorized person |
| **Fee** | Free |

Paper filing **not accepted**.

---

## Accounts Filing

### When Accounts Must Be Filed

| Income | Requirement |
|--------|-------------|
| **Under £25,000** | Not required (but recommended) |
| **£25,000 - £1m** | File with annual return |
| **Over £1m** | File within 10 months |

### Accounts Format

| Income | Format |
|--------|--------|
| **Under £250,000** | Receipts and payments OR accruals |
| **£250,000 - £1m** | Accruals (SORP) |
| **Over £1m** | Accruals (SORP) + audit |

---

## Serious Incident Reporting

### Must Report Immediately

| Incident Type | Examples |
|---------------|----------|
| **Fraud/theft** | By trustees, staff, volunteers |
| **Safeguarding** | Concerns about vulnerable beneficiaries |
| **Terrorism** | Links to proscribed organizations |
| **Criminal activity** | In charity's name |
| **Significant loss** | Of assets or data |

**Deadline**: As soon as reasonably possible (not annual return).

---

## Charitable Companies: Dual Filing

Charitable companies must file with **both**:

| Regulator | What to File | Deadline |
|-----------|--------------|----------|
| **Charity Commission** | Annual return + accounts | 10 months |
| **Companies House** | Statutory accounts | 9 months |

### Timing Note

Companies House deadline is **earlier** (9 months vs 10 months).

---

## CIOs: Charity Commission Only

CIOs file **only** with Charity Commission:
- No Companies House filing
- Same annual return requirements
- Same accounts requirements

---

## Penalties

### Failure to File

| Consequence | Detail |
|-------------|--------|
| **Warning** | First reminder from CC |
| **Second warning** | Formal request |
| **Double-defaulter** | Two years without return |
| **Removal from register** | Charity may be removed |
| **Trustee action** | Disqualification possible |

**No financial penalties** (unlike Companies House), but serious consequences.

---

## Updating During Year

### Changes to Report Promptly

| Change | How to Report |
|--------|---------------|
| **Trustee changes** | Update online (not annual return) |
| **Address change** | Update online |
| **Name change** | Application to CC |
| **Objects change** | Constitutional amendment |
